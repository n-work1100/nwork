Верхні рівні:
7. Прикладний (Application): Забезпечує взаємодію користувача та мережі (веббраузери, електронна пошта).
6. Представлення (Presentation): Перетворює формати, шифрує та стискає дані (наприклад, кодування ASCII, JPEG).
5. Сеансовий (Session): Керує з'єднаннями та встановлює сеанси між програмами.

Нижні рівні:
4. Транспортний (Transport): Відповідає за надійність, доставку даних від точки до точки та контроль помилок (протоколи TCP, UDP).
3. Мережевий (Network): Визначає маршрут передачі даних, оперує IP-адресами та пакетами (на цьому рівні працюють маршрутизатори/роутери).
2. Канальний (Data Link): Організовує зв'язок між вузлами та передає дані кадрами (frames), оперує MAC-адресами (комутатори/свічі).

1. Фізичний (Physical): Передає потік бітів через середовище передачі (кабелі, радіохвилі, оптоволокно)

--1--
Зайти в привелігований режим
__________________________________________________________________________
R1> enable
__________________________________________________________________________


--2--
Зайти в режим глобальної конфігурації
__________________________________________________________________________
R1# configure terminal
__________________________________________________________________________


--3--
Подивитись конфиг який працює в даний момент
Подивитись конфиг з яким завантажується пристрій
__________________________________________________________________________
R1# show running-config
R1# show startup-config
__________________________________________________________________________


--4--
Задати ім'я пристрою
__________________________________________________________________________
R1(config)# hostname R1
__________________________________________________________________________


--5--
Зберегти налаштування
__________________________________________________________________________
R1# write
	*copy running-config startup-config
__________________________________________________________________________


--6--
Перезавантажити пристрій
__________________________________________________________________________
R1# reload
__________________________________________________________________________


--7--
Зайти на інтерфейс gigabitEthernet 0/0
__________________________________________________________________________
R1(config)# interface gigabitEthernet 0/0
__________________________________________________________________________


--8--
Дата IP для інтерфейса gigabitEthernet 0/0
__________________________________________________________________________
R1(config-if)# ip add 10.10.10.254 255.255.255.0
__________________________________________________________________________


--9--
Увімкнути/Вимкнути інтерфейс
__________________________________________________________________________
R1(config-if)# no shutdown
R1(config-if)# shutdown
__________________________________________________________________________


--10--
Подивитись ір адреси інтерфейсів
__________________________________________________________________________
R# show ip interface brief
__________________________________________________________________________


--11--
Скинути конфігурацію до заводської
__________________________________________________________________________
R# erase startup-config
__________________________________________________________________________


--12--
Поставити пароль '1100' на роз'єм
__________________________________________________________________________
R1(config)# line console 0
R1(config-line)# password 1100
R1(config-line)# login
__________________________________________________________________________


--13--
Поставити пароль '1100' на привилегійований режим
__________________________________________________________________________
R1(config)# enable secret 1100
__________________________________________________________________________


--14--
Поставити пароль '1100' на VTY(віддалене з'єднання)
_____________________________________
R1(config)# line vty 0 15
R1(config-line)# password 1100
R1(config-line)# login
__________________________________________________________________________


--15--
Зашифрувати паролі
__________________________________________________________________________
R1(config)#service password-encryption
__________________________________________________________________________


--16--
Створити банерне повідомлення дня
__________________________________________________________________________
R1(config)# banner motd #Authorized Access Only#
__________________________________________________________________________


--17--
Подивитись інформацію про конкретний інтерфейс
__________________________________________________________________________
R1# show interface g0/0
__________________________________________________________________________


--18--
Подивитись інформацію про ір інтерфейсів
__________________________________________________________________________
R1# show ip interface brief
__________________________________________________________________________


--19--
Подивитися маршрути роутера
__________________________________________________________________________
R1# show ip route
__________________________________________________________________________


--20--
Вказати маршрут роутеру з викиданням трафіку на його інтерфейс
__________________________________________________________________________
R1(config)# ip route 20.20.20.0 255.255.255.0 gigabitEthernet 0/0
__________________________________________________________________________


--21--
Задати IPv6
__________________________________________________________________________
R1(config)# ipv6 unicast-routing
R1(config)# ipv6 route 2001:db8:acad:1::/64 2001:db8:acad:2::2
__________________________________________________________________________


--22--
Вказати маршрут роутеру по замовчуванню
__________________________________________________________________________
Router(config)# ip route 0.0.0.0 0.0.0.0 {ip-address | exit-intf}
__________________________________________________________________________


--23--
Вказати маршрут IPv6 роутеру по замовчуванню
__________________________________________________________________________
Router(config)ipv6 route ::/0 2001:db8:a:1::1
__________________________________________________________________________


--24--
Показати список віланів(VLAN)
__________________________________________________________________________
R1# show vlan brief
__________________________________________________________________________


--25--
Створити Vlan 10 з іменем Office
__________________________________________________________________________
S1(config)# vlan 10
S1(config-vlan)# name Office
S1(config)# end
__________________________________________________________________________


--26--
Налаштувати VLAN для intукафсу f0/4
__________________________________________________________________________
S1(config)#int f0/4
S1(config-if)#switchport mode access 
S1(config-if)#switchport access vlan 10
__________________________________________________________________________


--27--
Зробити порт транковим(магістральним)
Дати дозвіл на проходження всім VLAN(тільки VLAN 10)
__________________________________________________________________________
S4(config-if)# switchport mode trunk
S4(config-if)# switchport mode trunk allowed vlan all (10 замість all)
S4(config-if)# no shutdown
__________________________________________________________________________


--28--
Подивитися список магістралів(trunk)
__________________________________________________________________________
S4# show interface trunk
__________________________________________________________________________


--29--
Для встановлення довіреного стану інтерфейсу використовується команда режиму конфігурації інтерфейсу
!!!Голосовий трафік!!!
__________________________________________________________________________
S3(config-vlan)# vlan 150
S3(config-vlan)# name VOICE

S3(config-if)# mls qos trust cos
S3(config-if)# switchport voice vlan 150
